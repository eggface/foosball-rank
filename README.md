# foosball-rank
Foosball Rank Record

Small project to show Foosball rankings.
Support 1 vs 1, 1 vs 2 and 2 vs 2.

*Based on:*
* Meteor 1.4
* ES6

*Meteor Packages:*
* reactive-table
* twbs:bootstrap
* highcharts (installed via NPM)
